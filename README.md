# Momentum V6

Sistema operativo pessoal mobile-first.

## Inclui
- execução diária e contingências
- semana editável e horários recomendados
- agenda por data
- ciclo de 12 semanas
- metas mensuráveis
- progresso e revisões
- exportação/importação de dados

## Desenvolvimento
```bash
npm install
npm run dev
npm run build
```
